# # Online Railway Reservation System

A Python and SQLite-based Online Railway Reservation System that allows users to register, log in, view available trains, book tickets, and view booking history.

## Features

- User Registration
- User Login
- View Available Trains
- Book Railway Tickets
- View Booking History
- SQLite Database Integration
- Simple Command Line Interface (CLI)

## Technologies Used

- Python
- SQLite
- Flask (API)
- SQL

## How to Run

1. Install Python.
2. Install dependencies.

```bash
pip install -r requirements.txt
```

3. Run the project.

```bash
python main.py
```
